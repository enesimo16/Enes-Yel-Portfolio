## 🚀 Özellikler

| Özellik | Detay |
|---|---|
| **SPA Router** | Hash tabanlı, sayfa yenilemesiz gezinme |
| **PWA** | Offline çalışma, mobil/masaüstü kurulabilir |
| **Çoklu Dil** | TR / EN anlık geçiş |
| **Dark / Light Tema** | LocalStorage ile kalıcı tercih |
| **Renk Teması** | 4 farklı renk paleti (Mor, Yeşil, Turuncu, Mavi) |
| **Typewriter Efekti** | Ana sayfada dönen başlık animasyonu |
| **Scroll Reveal** | Kaydırma ile beliren içerikler |
| **3D Tilt Efekti** | Fareye duyarlı proje kartları |
| **Canvas Arka Plan** | Etkileşimli network/particle efekti |
| **Hava Durumu API** | Open-Meteo ile canlı hava durumu |
| **Terminal CLI** | `help`, `about`, `projects`, `social` ve daha fazlası |
| **Not Defteri** | LocalStorage tabanlı CRUD not uygulaması |
| **Toast Bildirimleri** | Anlık kullanıcı geri bildirimleri |
| **Ses Efektleri** | Etkileşimli ses deneyimi |
| **Custom Cursor** | Özel fare imleci |
| **Skeleton Loading** | Fetch süresince modern yükleme ekranı |
| **Konami Code** | `e-n-e-s` yazınca konfeti 🎉 |
| **Print Modu** | `Ctrl+P` ile CV çıktısı |
| **EmailJS** | İletişim formu gerçek e-posta gönderimi |
